1.  下载最新的ILRuntime
     git clone https://github.com/Ourpalm/ILRuntime.git
2.  拷贝ILRunetime 目录进入 U3D工程Assets目录里， 并把bin obj 这些无用的目录删除
3. 拷贝Mono.Cecil库 
    3.1 进入 Mono.Cecil 目录编译 根据 u3d 工程选择对应的 framework readly only
    3.2 将编译好的dll 放入U3D 中的Assets目录里
    3.3 文件包括 ILRuntim.Mono.Cecil.dll, ILRuntime.Mono.Cecil.Mdb.dll，ILRuntime.Mono.Cecil.Pdb.dll和所有扩展名未PDB的文件
4. Mono.Cecil库也可从ILRuntimeDemo工程中拷贝编译好的文件

5. 建立hotfix 热更新工程
添加引用:
5.1. C:\Program Files\Unity2017\Editor\Data\Managed\UnityEngine\UnityEngine.dll，
5.2. C:\Program Files\Unity2017\Editor\Data\Managed\UnityEngine\UnityEngine.CoreModule.dll
5.3 D:\work\github\ILRuntimeU3D\ILRuntimeDemo\Library\ScriptAssemblies\Assembly-CSharp.dll
   保证hotfix工程能访问 u3d工程中的代码
5.4 设置使用的.Net Framework版本 要与 Assembly-CSharp.dll 一致
